# Calculadora em Java utilizando conceitos de POO e interface gráfica usando JavaFX

Calculadora desenvolvida em Java utilizando JavaFX, com aplicação do padrão arquitetural MVC (Model-View-Controller), separação em camadas e persistência de dados em arquivo. O projeto tem como foco a aplicação prática de conceitos de Programação Orientada a Objetos (POO) e organização de software.
